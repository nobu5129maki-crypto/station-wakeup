マイク・通知・オフライン用の設定は、上の「開き方」のとおり http://localhost:… または https://… で開いてください。

※ Android でホーム画面に追加したアプリから使う場合、画面内だけの振動が効かず、通知の振動頼みになることがあります。監視開始時の「通知を許可」は許可するか、Android の設定でこのアプリ（サイト）の通知をオンにしてください。マナーモードや「振動オフ」だと鳴りません。

※ iPhone の Safari では Web の振動 API に非対応のため、短いアラーム音でお知らせします（監視開始ボタンで音声の準備をします）。
